# Data Professional Survey Dashboard
This project is a Power BI dashboard analyzing survey data from data professionals around the world.

## Project Background
This dashboard was created based on online tutorial and further modified with additional visualizations, layout improvements, and data cleaning adjustments.

## Key Insights
• Python is the most popular programming language among data professionals.  
• Most respondents work in the technology industry.
• Job satisfaction is highest for coworkers and work/life balance. 
• Salary satisfaction tends to be lower compared to other job factors.  

## Dashboard Features
- Respondents distribution by country
- Job title and education level analysis
- Industry distribution
- Favorite programming languages
- Job satisfaction overview
- Difficulty breaking into the data field

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization
